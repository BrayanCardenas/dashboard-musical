# Music Dashboard

Un sistema para gestionar repertorio musical, músicos y eventos de forma profesional.  
Construido con **React + TypeScript + TailwindCSS**.

---

## Características

-  **Login** con formulario estilizado
-  **Dashboard** con métricas y próximos eventos
-  **Canciones**: administra tu repertorio con dificultad y género
-  **Músicos**: gestiona tu equipo con contacto y estado
-  **Agenda**: organiza eventos con fecha, lugar y estado
-  **Configuración**: ajusta perfil y preferencias
